Dino Game
---------
So, uhh, yeah. Well, I think we all know what this is: The Google Dinosaur Game
recoded for Windows and python, so this is available on any platform that supports
python. Yeah! Even Windows! That means that you can play this game on Windows, macOS,
Linux, BSD, Android, and even on iOS using a-shell!

System Requirements:
--------------------
-Python 3.14
-Pygame installed (recommended to use Pygame-ce)
-Kivy installed (a-shell on iOS)

Downloads:
----------
-Download Python Here:
https://www.python.org/downloads/

-(i) please note that python is already installed on a-shell for iOS
you can download and install pygame using this command:
pip install pygame-ce

Other Information
-----------------
Please note that you can't run the python version directly, you have to run it using the terminal. However, we are
doing our best to fix that.
